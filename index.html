Thoughts sharing - Django Based App

×

It is Django based app, where users can login to create, read articles give comments on others articles.

Skills

  • Server Side Rendering
  • JavaScript + Django for getting Async behavior

Authentication

  • Login, Logout, Reset Email
  • Built using Class-Based views.

Functionalities

  • Create, edit, view, delete -> articles & review
  • Admin panel to have control over users articles and comments
  • Searching functionality for articles & users
  • User profile tailored to each user, displays user's profile information, articles, reviews, and favorites.
  • Html text rich editor to make it easier for users to create good articles
  • Option to add articles to user's favorites
  • Pagination using paginator

Technologies & Implementations

  • Database: relationship between different models
  • Both class-based & function-based Views
  • Connecting aws bucket with heroku
  • Different permissions defined so a user can have access only to his data

Integrations and Deployment

  • TinyMCE editor is integrated for having Html text rich editor
  • AWS 3 is used for storing media files
  • The Project is deployed on Heroku

Aurora Store is a fictional online shopping website built using React, Redux, Routing, Firebase and much more.

  • Searching products from all the products currently available

Technologies used and Implementations

  • React for UI
  • Redux for state management
  • Firebase for authentication and storing cart data of a user
  • React router for routing between different pages
